# Reinforcement Learning Labs
### M.Tech AI | IIT Kharagpur

This repository contains implementations of RL algorithms, covering fundamental bandit problems and Deep Q-Networks.

## Project Jist
- **[Multi-Armed Bandits: Exploration-Exploitation Trade-off](./01-Multi-Armed-Bandits)**: A study on regret minimization in stochastic environments using classical bandit algorithms.
- **[Deep Q-Networks (DQN) for Discrete Control](./02-Deep-Q-Networks)**: Implementing a stable DQN agent to solve the Gymnasium CartPole-v1 control task.
